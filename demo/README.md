# Demo lesson — for showing Quest's ingest → quest flow

A realistic teaching document a teacher would upload to Quest. The content follows the
real Vietnamese curriculum (**Chương trình GDPT 2018, Lịch sử và Địa lí lớp 6**), topic
*"Bước ngoặt lịch sử đầu thế kỉ X — Ngô Quyền và chiến thắng Bạch Đằng năm 938."*

## Files
| File | Use in the demo |
|---|---|
| `bai-hoc-bach-dang-938.pdf` | Upload as a file (the realistic path: "I upload my lesson PDF"). |
| `bai-hoc-bach-dang-938.html` | Source for the PDF; also fine to open/print. |
| `bai-hoc-bach-dang-938.txt` | Paste into the "dán văn bản" box (fastest path, no file). |

## How to run it
1. As Người dẫn đường, create a class and add this as a material (upload the PDF, or paste the .txt).
2. Build a Nhiệm vụ from it. Optional **yêu cầu thêm** to the AI, e.g.:
   > *"Tập trung vào mốc thời gian và kế sách cọc Bạch Đằng; ra câu hỏi bằng tiếng Việt đơn giản, có vài câu tình huống."*
3. Send the quest to a learner and run the 5-minute adaptive quiz.

## What good output looks like
The lesson is built to decompose into ~5 concepts, so per-concept mastery (BKT) and
difficulty-matching are visible in the demo:
- Họ Khúc giành quyền tự chủ (905)
- Dương Đình Nghệ chống Nam Hán (931)
- Hoàn cảnh trước trận Bạch Đằng (Kiều Công Tiễn, 937–938)
- Kế sách & diễn biến trận Bạch Đằng (cọc ngầm, thủy triều, 938)
- Ý nghĩa lịch sử (chấm dứt Bắc thuộc, Ngô Quyền xưng vương 939)

Clear dates and cause→effect make it easy to generate MCQs across easy/medium/hard.
